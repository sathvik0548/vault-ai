Vault-AI: Operating Manual

System Requirements

  • OS: MacOS (Intel/ARM), Linux, Windows
  • Runtime: Python 3.9+
  • Minimum RAM: 2GB

Remote Server Sync Deployment

Vault-AI uses a standard HTTP model for remote backups.

python3 server.py 0.0.0.0 8000
python3 cli.py sync --url http://YOUR-IP:8000

AI Configuration

Vault-AI supports local and remote Large Language Models (LLMs).

python3 cli.py setup

Supported Providers: Ollama, Gemini, OpenAI. Keys are stored locally in .vault/config.json.
