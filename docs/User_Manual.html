Vault-AI: User Manual

Welcome to Vault-AI, the AI-native, human-centric Version Control System. This manual explains how to use Vault-AI effectively for your daily projects.

Core Concepts

Unlike Git, Vault-AI utilizes Zero-Staging. When you want to save a change, we capture the state of your entire working tree safely without requiring you to "add" each file individually.

Initialization

Before tracking code, initialize your vault in your project repository:

python3 cli.py init

Saving Changes

When you have finished making changes to your code, execute the save command:

python3 cli.py save

Our AI Brain will automatically scan the code differences between your last save and current state, intelligently writing a highly-semantic commit message for you.

Viewing History and Status

python3 cli.py history
python3 cli.py status

Sandboxes

Sandboxes are isolated testing environments that do not break your primary codebase.

python3 cli.py sandbox enter
python3 cli.py sandbox merge
